## 문서 작성 가이드라인

### Markdown 문서 구조

모든 `.md` 문서는 다음 구조를 따릅니다:

```markdown
# {문서 제목}

## {주요 섹션}
- 표 형식으로 정리된 내용
- 시간, 안건, 세부 내용 포함

---

## 준비물
체크리스트 형식

---

## 다음 단계 (Action Items)
액션 아이템 테이블
```

### 표 형식 활용

정보는 가급적 표 형식으로 정리:

```markdown
| 항목 | 내용 |
|------|------|
| 일시 | 2026년 월 일 (  ) 00:00 ~ 00:00 |
| 장소 | |
| 참석자 | 실무 직원 |
```
