### 2.2 업무별 AI 활용 방법

#### A. 고객 응대 업무

**활용 시나리오**

| 상황 | AI 활용 방법 | 예시 프롬프트 |
|------|-------------|--------------|
| FAQ 답변 작성 | 표준 답변 초안 생성 | "여행사 고객이 자주 묻는 환불 관련 질문에 대한 친절한 답변을 작성해줘" |
| 클레임 응대 | 사과 및 해결 제안 작성 | "항공편 지연으로 불만을 제기한 고객에게 보낼 사과 이메일 초안을 작성해줘" |
| 상담 스크립트 | 상황별 대응 시나리오 | "첫 해외여행 고객을 위한 상담 스크립트를 작성해줘" |

**주의사항**
- 고객 개인정보(이름, 연락처, 예약번호 등)는 절대 입력하지 않습니다
- AI 생성 답변은 반드시 검토 후 사용합니다
- 회사 정책과 일치하는지 확인합니다

#### B. 문서 작성 업무

**활용 시나리오**

| 상황 | AI 활용 방법 | 예시 프롬프트 |
|------|-------------|--------------|
| 제안서 초안 | 구조 및 내용 생성 | "관광지원과에 제출할 단체여행 제안서 개요를 작성해줘" |
| 보고서 작성 | 데이터 정리 및 분석 | "월간 예약 현황을 요약하는 보고서 형식을 만들어줘" |
| 이메일 작성 | 비즈니스 이메일 초안 | "항공사 담당자에게 단체 좌석 협조를 요청하는 공식 이메일을 작성해줘" |

**효과적인 프롬프트 작성 팁**

```
[구조]
1. 역할 부여: "당신은 여행사 제안서 전문가입니다"
2. 구체적 요청: "A4 2페이지 분량의..."
3. 조건 명시: "공식적인 톤으로, 숫자는 표로 정리해서"
4. 출력 형식: "제목, 본문, 결론 순서로"
```

#### C. 프레젠테이션 제작

**활용 시나리오**

| 단계 | AI 활용 방법 |
|------|-------------|
| 구조 설계 | 슬라이드 구성 제안 받기 |
| 내용 작성 | 각 슬라이드별 핵심 메시지 생성 |
| 디자인 | Gamma 등 AI 도구로 자동 디자인 |
| 검토 | 내용 검토 및 개선점 피드백 |

**예시 프롬프트**
```
"관공서에 제출할 단체여행 제안 프레젠테이션을 만들어줘.
- 대상: 지자체 공무원 30명
- 여행지: 제주도 2박3일
- 포함 내용: 일정, 숙소, 식사, 예산
- 슬라이드 수: 10장 내외
- 톤: 신뢰감 있고 전문적인"
```

### 2.3 AI 활용 시 주의사항

#### 정보 보안 가이드라인

**절대 입력 금지 정보**

| 구분 | 예시 |
|------|------|
| 고객 개인정보 | 이름, 연락처, 여권번호, 카드번호 |
| 회사 기밀정보 | 매출 데이터, 계약 조건, 내부 전략 |
| 비공개 가격정보 | 원가, 마진율, 특별 할인율 |

**안전한 사용 원칙**

```
1. 가명 또는 예시 데이터 사용
2. 민감 정보 마스킹 (예: 홍** 010-****-1234)
3. 결과물 검토 후 사용
4. 의심스러운 경우 상급자 문의
```

#### AI 결과물 검증

| 검증 항목 | 확인 사항 |
|----------|----------|
| 정확성 | 사실관계, 숫자, 날짜 확인 |
| 일관성 | 회사 정책/가이드라인 부합 여부 |
| 적절성 | 톤, 표현, 형식의 적합성 |
| 완전성 | 빠진 내용이 없는지 확인 |
